第2章. 振幅と確率はどうつながるのか？

第1章では、キュービット状態を |ψ⟩ = α|0⟩ + β|1⟩ のように書く方法を見ました。
この章では、その状態を測定するとき、振幅 α と β が確率としてどう読まれるのかを見ていきます。

1段階

状態の中では振幅を読みます

キュービット状態を書くとき、α と β は確率ではありません。
α は |0⟩ 方向に付いた振幅で、β は |1⟩ 方向に付いた振幅です。

|ψ⟩ = α|0⟩ + β|1⟩
2段階

測定するとき、振幅を確率として読み替えます

測定結果の確率は、振幅そのものをそのまま読んで得るわけではありません。
それぞれの振幅の絶対値を2乗して、確率を計算します。

α → |α|² , β → |β|²
3段階

2つの確率を足すと100%になります

1つのキュービットを測定すると、測定結果0または測定結果1のどちらかが出ます。
したがって2つの測定確率を足すと、必ず1、つまり100%になります。

|α|² + |β|² = 1

正確に言うと、α や β のような値は確率振幅です。
ただし説明が重くなりすぎないように、この本ではふつう振幅と呼びます。

測定結果0の確率 = |α|²、測定結果1の確率 = |β|²

この規則をBorn規則と呼びます。
振幅が実数ならふつうの2乗として読んでもよいですが、一般の量子力学では振幅が複素数になることもあるため、絶対値の2乗を使います。

状態の中で読む対象状態の中に入っている値測定すると得る値
|0⟩ 方向振幅 α確率 |α|²
|1⟩ 方向振幅 β確率 |β|²

正規化という名前も付けておきます

2つの測定確率を足すと1、つまり100%になることを見ました。
この条件を満たす状態を正規化された状態と呼びます。
式では次のように書きます。

|α|² + |β|² = 1

今は「全体の測定確率が100%になるように整えられた状態」と理解すれば十分です。

50%の確率なのに振幅が1/√2になるのはなぜか？

測定結果0と測定結果1が半々なら、振幅もそれぞれ 1/2 のように見えるかもしれません。
しかし確率は振幅の2乗から出るので、50%の確率を作るには振幅の大きさが 1/√2 になります。

誤った考え

振幅を1/2にするとどうなるか？

各測定結果の確率は25%になります。

(1/2)² + (1/2)² = 1/4 + 1/4 = 1/2

全体の確率が50%にしかならないため、これは正しいキュービット状態ではありません。

正しい振幅

振幅を1/√2にするとどうして合うのか？

各測定結果の確率が50%になります。

(1/√2)² + (1/√2)² = 1/2 + 1/2 = 1

全体の確率も100%になります。

Qoraで試してみよう

キュービットを1つ準備して測定してみよう

operation Main() {
    // キュービットを1つ準備
    // 準備直後の q[0] は |0⟩ 状態
    // 測定結果0の振幅 = 1
    // 測定結果1の振幅 = 0
    use q = Qubit[1];

    // q[0] を測定
    // 測定は現在の振幅を確率として読み、結果を1つ得る段階
    // 理想的な実行環境: 測定結果0
    var result: bit = M(q[0]);
}
