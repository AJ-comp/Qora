第1章. キュービット状態と振幅について見ていこう

キュービット状態を最初から難しく考える必要はありません。
まずは「測定結果をどう状態の方向として書くのか」「その方向の前にどうして数が付くのか」を順番に見ていきます。

1段階

測定結果に状態方向の名前を付けます

1つのキュービットを測定すると、結果は0または1になります。
この結果をただの数字 0、1 と書くこともできそうですが、それだけでは「測定結果」という意味は見えても、状態ベクトルの基準方向だという意味は見えにくくなります。
そこで量子力学では、1キュービット状態をベクトルとして扱えるように、測定結果0に対応する基底状態を |0⟩ と書き、測定結果1に対応する基底状態を |1⟩ と書きます。

測定結果0 → |0⟩
0側の状態方向
測定結果1 → |1⟩
1側の状態方向

この2つの記号 |0⟩ と |1⟩ を、1キュービットの計算基底と呼びます。

2段階

基準方向の前には、その成分の大きさを表す数が付きます

先ほど |0⟩ と |1⟩ を状態ベクトルの基準方向と呼びました。
まだ少しなじみにくいので、まずは平面上のふつうのベクトルと同じ構造を見てみましょう。
平面ベクトルは、x軸方向の成分とy軸方向の成分に分けて書けます。

3·x方向 + 2·y方向

ここで3はx軸方向成分の係数で、2はy軸方向成分の係数です。
つまりベクトルを基準方向ごとに分けて書くには、それぞれの方向の前に「その方向の成分がどれだけあるか」を表す数が付きます。

3段階

キュービット状態も基準方向ごとの成分として書きます

キュービット状態では、|0⟩ と |1⟩ が基準方向の役割を持ちます。
そのため各方向の前に数が必要になり、量子力学ではその数を振幅と呼びます。

α|0⟩ + β|1⟩
α |0⟩ 方向の成分がどれだけ入っているかを表す振幅です。
β |1⟩ 方向の成分がどれだけ入っているかを表す振幅です。
|ψ⟩ = α|0⟩ + β|1⟩

したがってこの式は、「このキュービット状態には0側の振幅 α と1側の振幅 β が一緒に入っている」という意味です。

ここで |ψ⟩ は、このキュービットの状態全体を表す記号です。
ψ はギリシャ文字のプサイで、|ψ⟩ はふつうケット・プサイまたはプサイ状態と読みます。
正式には、|ψ⟩ は |0⟩ 方向の振幅と |1⟩ 方向の振幅を一緒に含む状態ベクトルです。
つまりキュービット状態は、「あとで0が出るのか1が出るのか」という答え1つではなく、可能な測定結果ごとに付いた振幅全体を持っています。

|0⟩ と |1⟩ は計算基底状態です |0⟩ は測定すると必ず測定結果0が出る基底状態で、|1⟩ は測定すると必ず測定結果1が出る基底状態です。
今はこの2つを「0側の方向」と「1側の方向」と読めれば十分です。
数学的には、|0⟩ と |1⟩ は1キュービット状態空間の互いに直交する2つの基準ベクトルとして働きます。
この章で押さえたこと 1キュービット状態は、|0⟩ 方向の成分と |1⟩ 方向の成分を一緒に持つ状態ベクトルです。
そして α と β は各方向成分の前に付く数で、量子力学ではこの数を振幅と呼びます。
