Chapter 15. Read a Small Circuit All the Way Through

Now it's time to read one short circuit from beginning to end.
This circuit prepares two qubits, applies an H gate to the first qubit, and then connects the two qubits with a CNOT gate.
It is the Bell state circuit, which you will often meet in introductions to quantum computing.

Try It in Qora

Let's Write the Bell State Circuit in Qora

operation Main() {
    // Prepare two qubits
    // Right after preparation, q[0]q[1] is in the |00⟩ state
    use q = Qubit[2];

    // Apply an H gate to the first qubit
    // Amplitudes appear for |00⟩ and |10⟩
    H(q[0]);

    // Apply the CNOT gate
    // The target qubit on the |10⟩ side flips, giving |11⟩
    CNOT(q[0], q[1]);

    // Measure both qubits
    // In an ideal execution environment: 00 or 11
    bit first = M(q[0]);
    bit second = M(q[1]);
}

Let's Follow the Circuit as State Changes

StepStateHow to read it
Right after preparation|00⟩Both qubits sit on the measurement outcome 0 side
After H(q[0])(|00⟩ + |10⟩) / √2The first qubit's amplitude is split between 0 and 1
After CNOT(|00⟩ + |11⟩) / √2On the branch where the control qubit is 1, the target qubit flips
Measurement00 or 11The two outcomes come out locked together

The key to reading this circuit is not "we measure the first qubit to get 0 or 1, and then change the second qubit."
Instead, read it as: the two amplitude branches created by the H gate pass through the CNOT gate and settle into a |00⟩ branch and a |11⟩ branch.
Then, when we measure at the end, the two qubits' outcomes appear together as 00 or 11.

X gate

Swaps the positions of amplitudes

It swaps the amplitude of measurement outcome 0 with the amplitude of measurement outcome 1.

H gate

Splits and distributes amplitude

It splits the amplitude that was on one side across both measurement outcomes, setting up interference later.

Z gate

Changes the sign

It flips the sign of the amplitude of measurement outcome 1, and that difference shows up in the next interference.

CNOT gate

Connects two qubits

It flips the target qubit only on the branch where the control qubit is 1, creating a relationship between the two qubits.
