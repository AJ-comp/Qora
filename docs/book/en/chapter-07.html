Chapter 7. How Do We Read a Two-Qubit State?

With one qubit, the possible measurement outcomes were 0 and 1.
With two qubits, a measurement outcome is no longer a single digit. It is a two-bit string.
In this document, we read the left digit as the first qubit and the right digit as the second qubit.

The new notation looks bigger, but the idea is still simple One qubit has two basis states.
Two qubits have four basis states.
So the state vector holds four amplitudes instead of two.

The computational basis has four states

Measuring two qubits can give 00, 01, 10, or 11.
So we write four computational-basis states for the two-qubit system.

Basis stateFirst qubitSecond qubitMeasurement result
|00⟩0000
|01⟩0101
|10⟩1010
|11⟩1111

The state vector holds four amplitudes

If a one-qubit state can be written as α|0⟩ + β|1⟩, a two-qubit state is written by attaching amplitudes to the four basis states.

|ψ⟩ = a₀₀|00⟩ + a₀₁|01⟩
+ a₁₀|10⟩ + a₁₁|11⟩

Here, a₀₀ is the amplitude attached to measurement outcome 00, and a₀₁ is the amplitude attached to 01.
In the same way, a₁₀ belongs to 10, and a₁₁ belongs to 11.
Measurement probabilities still come from squared absolute values of amplitudes.

Measurement outcomeAmplitudeMeasurement probability
00a₀₀|a₀₀|²
01a₀₁|a₀₁|²
10a₁₀|a₁₀|²
11a₁₁|a₁₁|²

The total measurement probability must still add up to 100%.

|a₀₀|² + |a₀₁|² + |a₁₀|² + |a₁₁|² = 1

A two-qubit state is not always just two separate states pasted together

Some two-qubit states can be described by writing the first qubit state and the second qubit state separately, then combining them.
But not every two-qubit state can be separated that way.
That difference is the starting point for entanglement in the next chapter.

Try It in Qora

Prepare two qubits

operation Main() {
    // Prepare two qubits
    // Right after preparation, the full state is |00⟩
    // Amplitude of measurement outcome 00 = 1
    // Amplitudes of the other outcomes = 0
    use q = Qubit[2];

    // Measure the first qubit
    var first: bit = M(q[0]);

    // Measure the second qubit
    var second: bit = M(q[1]);
}
The question for the next chapter Among two-qubit states with four amplitudes, some cannot be fully explained by separating the first qubit from the second qubit.
Those states are called entangled states.
