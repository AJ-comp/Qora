Let's Get Ready to Read Quantum

In the previous chapter, we wrote a qubit state as amplitudes attached to the |0⟩ direction and the |1⟩ direction.
Now we will keep that state picture and ask what those amplitudes become when the qubit is measured.

Chapter 2. How Are Amplitudes and Probabilities Connected?

In Chapter 1, we wrote a qubit state as |ψ⟩ = α|0⟩ + β|1⟩.
In this chapter, we'll see how the amplitudes α and β are read as probabilities when we measure the state.

Step 1

Inside the state, we read amplitudes

When we write a qubit state, α and β are not probabilities.
α is the amplitude attached to the |0⟩ direction, and β is the amplitude attached to the |1⟩ direction.

|ψ⟩ = α|0⟩ + β|1⟩
Step 2

When we measure, amplitudes are converted into probabilities

We do not get a measurement probability by reading the amplitude itself.
We calculate the probability by taking the squared absolute value of each amplitude.

α → |α|² , β → |β|²
Step 3

The two probabilities must add up to 100%

Measuring one qubit gives either measurement outcome 0 or measurement outcome 1.
Therefore the two measurement probabilities must add up to 1, which means 100%.

|α|² + |β|² = 1

More precisely, values such as α and β are probability amplitudes.
To keep the wording lighter, this book usually calls them simply amplitudes.

Probability of measurement outcome 0 = |α|², probability of measurement outcome 1 = |β|²

This rule is called the Born rule.
If the amplitudes are real numbers, you can read this as ordinary squaring, but in general quantum mechanics amplitudes can be complex, so we use squared absolute values.

What we read in the stateValue inside the stateValue obtained when measured
|0⟩ directionamplitude αprobability |α|²
|1⟩ directionamplitude βprobability |β|²

We also give this condition a name: normalization

We just saw that the two measurement probabilities must add up to 1, or 100%.
A state that satisfies this condition is called a normalized state.
In symbols, we write the condition like this.

|α|² + |β|² = 1

For now, it is enough to understand this as “a state whose total measurement probability is correctly set to 100%.”

Why is the amplitude for 50% probability 1/√2?

If measurement outcome 0 and measurement outcome 1 are split evenly, it may feel natural to set each amplitude to 1/2.
But probabilities come from squared amplitudes, so to make a 50% probability, the amplitude size must be 1/√2.

Wrong idea

What happens if each amplitude is 1/2?

Each measurement outcome gets probability 25%.

(1/2)² + (1/2)² = 1/4 + 1/4 = 1/2

The total probability is only 50%, so this is not a valid qubit state.

Correct amplitude

Why does 1/√2 work?

Each measurement outcome gets probability 50%.

(1/√2)² + (1/√2)² = 1/2 + 1/2 = 1

The total probability is also 100%.

Try It in Qora

Prepare one qubit and measure it

operation Main() {
    // Prepare one qubit
    // Right after preparation, q[0] is in the |0⟩ state
    // Amplitude of measurement outcome 0 = 1
    // Amplitude of measurement outcome 1 = 0
    use q = Qubit[1];

    // Measure q[0]
    // Measurement turns the current amplitudes into one outcome
    // In an ideal execution environment: measurement outcome 0
    var result: bit = M(q[0]);
}
