Chapter 9. Quantum Gates Must Be Reversible.

So far, we have watched gates change the amplitudes inside a qubit state.
But not every rule that changes a state can be a quantum gate.
The first condition to keep in mind is this: before measurement, a quantum gate must not crush information and throw it away.

Step 1 Different inputs must go to different outputs. (orthogonality preservation)
If two input states merge into the same output state, the output alone cannot tell us which input we started from.
For example Think about a rule that keeps only the result of addition.
(2, 3) goes to 5, (3, 2) also goes to 5, and (1, 4) also goes to 5.
From the output 5 alone, we cannot know whether the original input was (2, 3), (3, 2), or (1, 4).
Several inputs have been merged into one output.
Step 2 The total probability must stay 100%. (length preservation)
If the length of the state vector changes, the measurement probabilities no longer add up correctly.
For example Suppose a rule sends |0⟩ to |0⟩ + |1⟩.
Then the probability of measurement outcome 0 is |1|² = 1, and the probability of measurement outcome 1 is also |1|² = 1.
The total probability becomes 1 + 1 = 2, or 200%.
So when amplitude is sent toward two directions, its size must be adjusted, as in (1/√2)|0⟩ + (1/√2)|1⟩, so the total probability stays 100%.
Step 3 The transformation must be reversible. (inverse exists)
Before measurement, a quantum gate is not a command that erases a state and overwrites it with a new one.
It moves the state vector inside state space, and there must be a reverse transformation that can bring the state back.
Example The simplest example is a rule that swaps the two states.
It sends |0⟩ to |1⟩, and |1⟩ to |0⟩.
Applying the same rule once more sends |0⟩ through |1⟩ and back to |0⟩, and sends |1⟩ through |0⟩ and back to |1⟩.
We will later call this swapping rule the X gate.

These three steps all point to the same idea.
Before measurement, a quantum gate must keep the input information, preserve the total probability, and allow us to follow the transformation backward.
That is why different input states must be sent to distinguishable output states.

Mathematically, this condition is summarized by the term unitary transformation.
In a standard quantum circuit, every gate before measurement must be unitary, and the condition is written like this.

U†U = I

Here U is the gate matrix, and U† is its conjugate transpose.
The symbols may feel unfamiliar at first, but for now it is enough to read this as “after applying the gate, we can undo it by going backward.”

The point to keep Before measurement, a quantum gate must not crush different input states into one output state.
It must also preserve the length of the state vector so the total probability remains 100%.
That is why we treat quantum gates as reversible transformations, or unitary transformations.

This principle will matter again when we study the H gate.
In particular, the reason the H gate needs a minus sign will be explained using this condition: different input states must not merge into the same output state.
