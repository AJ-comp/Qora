9장. 양자 게이트는 되돌릴 수 있어야 한다.

앞장까지는 게이트가 큐비트 상태의 진폭을 바꾸는 모습을 봤어요.
그런데 상태를 바꾸는 규칙이라고 해서 아무 규칙이나 양자 게이트가 될 수 있는 것은 아니에요.
먼저 가장 중요한 조건부터 잡아볼게요. 측정 전의 양자 게이트는 정보를 뭉개서 잃어버리면 안 돼요.

1단계 서로 다른 입력은 서로 다른 출력으로 가야 해요. (직교성 보존)
두 입력 상태가 같은 출력 상태로 합쳐지면, 출력만 보고 원래 입력을 되살릴 방법이 없어져요.
예를 들어 더하기 결과만 남기는 규칙을 생각해봐요.
(2, 3)도 5로 가고, (3, 2)도 5로 가고, (1, 4)도 5로 가요.
출력 5만 보면 원래 입력이 (2, 3)이었는지, (3, 2)였는지, (1, 4)였는지 알 수 없어요.
여러 입력이 하나의 출력으로 합쳐졌기 때문이에요.
2단계 전체 확률은 계속 100%로 남아야 해요. (길이 보존)
상태 벡터의 길이가 달라지면, 측정 확률의 합도 맞지 않게 돼요.
예를 들어 어떤 규칙이 |0⟩을 |0⟩ + |1⟩로 보낸다고 해볼게요.
그러면 측정 결과값 0의 확률은 |1|² = 1이고, 측정 결과값 1의 확률도 |1|² = 1이에요.
전체 확률이 1 + 1 = 2, 즉 200%가 되어버려요.
그래서 진폭을 두 방향으로 보낼 때는 (1/√2)|0⟩ + (1/√2)|1⟩처럼 전체 확률이 100%로 남도록 크기를 맞춰야 해요.
3단계 거꾸로 되돌릴 수 있어야 해요. (역변환 존재)
측정 전의 양자 게이트는 상태를 지워서 새 상태로 덮어쓰는 명령이 아니에요.
상태 벡터를 상태 공간 안에서 움직인 뒤, 반대 변환을 적용하면 원래 상태로 돌아올 수 있어야 해요.
예시 가장 단순한 예로 두 상태를 서로 바꾸는 뒤집기 규칙을 생각해봐요.
이 규칙은 |0⟩을 |1⟩로 보내고, |1⟩을 |0⟩으로 보내요.
같은 규칙을 한 번 더 적용하면 |0⟩은 |1⟩을 거쳐 다시 |0⟩으로 돌아오고, |1⟩은 |0⟩을 거쳐 다시 |1⟩로 돌아와요.
나중에 이 뒤집기 규칙을 X 게이트라고 부를 거예요.

방금 세 단계는 모두 같은 이야기로 이어져요.
측정 전의 양자 게이트는 입력 상태를 잃어버리지 않고, 전체 확률을 보존하면서, 다시 거꾸로 따라갈 수 있어야 해요.
그래서 측정 전의 양자 게이트는 서로 다른 입력 상태를 서로 구별 가능한 출력 상태로 보내야 해요.

이 조건을 수학적으로 정리하면 유니터리 변환이라는 말이 나와요.
표준 양자 회로에서 측정 전의 게이트는 유니터리 변환이어야 하고, 그 조건을 식으로 쓰면 아래처럼 적어요.

U†U = I

여기서 U는 게이트 행렬이고, U†는 그 행렬의 켤레전치예요.
기호는 낯설 수 있지만, 지금은 “게이트를 적용한 뒤 다시 거꾸로 되돌릴 수 있다”는 뜻으로 읽으면 충분해요.

조건뜻왜 필요한가
길이 보존상태 벡터의 전체 확률이 1로 유지돼요측정 확률의 합이 항상 100%여야 해요
직교성 보존서로 구별 가능한 입력 상태가 출력에서도 구별돼요서로 다른 입력이 같은 출력으로 합쳐지면 안 돼요
역변환 존재게이트를 거꾸로 되돌릴 수 있어요측정 전의 양자 진화는 정보를 잃지 않아요
여기서 잡고 갈 핵심 측정 전의 양자 게이트는 서로 다른 입력 상태를 하나의 출력 상태로 뭉개면 안 돼요.
또 전체 확률이 100%로 유지되도록 상태 벡터의 길이를 보존해야 해요.
그래서 양자 게이트는 되돌릴 수 있는 변환, 즉 유니터리 변환으로 다뤄요.

이 원칙은 뒤에서 H 게이트를 볼 때 다시 중요해져요.
특히 H 게이트의 마이너스 부호가 왜 필요한지는, 바로 이 “서로 다른 입력 상태가 같은 출력 상태로 합쳐지면 안 된다”는 조건으로 설명할 거예요.
