2장. 진폭과 확률은 어떻게 연결될까?

1장에서는 큐비트 상태를 |ψ⟩ = α|0⟩ + β|1⟩처럼 적는 법을 봤어요.
이번 장에서는 그 상태를 측정할 때, 진폭 α와 β가 확률로 어떻게 읽히는지 살펴볼 거예요.

1단계

상태 안에서는 진폭을 읽어요

큐비트 상태를 적을 때 α와 β는 확률이 아니에요.
α는 |0⟩ 방향에 붙은 진폭이고, β는 |1⟩ 방향에 붙은 진폭이에요.

|ψ⟩ = α|0⟩ + β|1⟩
2단계

측정할 때 진폭을 확률로 바꿔 읽어요

측정 결과값의 확률은 진폭을 그대로 읽어서 얻지 않아요.
각 진폭의 절댓값을 제곱해서 확률을 계산해요.

α → |α|² , β → |β|²
3단계

두 확률을 더하면 100%가 되어야 해요

한 큐비트를 측정하면 결과값 0 또는 결과값 1 중 하나가 나와요.
그래서 두 측정 확률을 모두 더하면 항상 1, 즉 100%가 되어야 해요.

|α|² + |β|² = 1

정확히 말하면 α와 β 같은 값은 확률 진폭이에요.
다만 앞으로 설명이 너무 무거워지지 않도록, 이 문서에서는 보통 진폭이라고 부를게요.

측정 결과값 0의 확률 = |α|², 측정 결과값 1의 확률 = |β|²

이 규칙을 Born 규칙이라고 불러요.
진폭이 실수일 때는 그냥 제곱으로 보아도 되지만, 일반적인 양자역학에서는 진폭이 복소수일 수 있으므로 절댓값 제곱으로 계산해요.

상태를 읽는 대상상태 안에 들어 있는 값측정할 때 얻는 값
|0⟩ 방향진폭 α확률 |α|²
|1⟩ 방향진폭 β확률 |β|²

정규화라는 이름도 붙여둬요

앞에서 두 측정 확률을 모두 더하면 1, 즉 100%가 되어야 한다고 봤어요.
이 조건을 만족하는 상태를 정규화된 상태라고 불러요.
수식으로는 아래처럼 적어요.

|α|² + |β|² = 1

지금은 “전체 측정 확률이 100%로 맞춰진 상태”라고 이해하면 충분해요.

50% 확률의 진폭은 왜 1/√2일까?

측정 결과값 0과 측정 결과값 1이 반반이면 진폭도 각각 1/2일 것처럼 보일 수 있어요.
하지만 확률은 진폭의 제곱에서 나오므로, 50% 확률을 만들려면 진폭의 크기가 1/√2이어야 해요.

틀린 생각

진폭을 1/2로 두면 어떻게 될까?

각 측정 결과값의 확률은 25%가 돼요.

(1/2)² + (1/2)² = 1/4 + 1/4 = 1/2

전체 확률이 50%밖에 되지 않으므로 큐비트 상태로 맞지 않아요.

올바른 진폭

진폭을 1/√2로 두면 왜 맞을까?

각 측정 결과값의 확률이 50%로 맞춰져요.

(1/√2)² + (1/√2)² = 1/2 + 1/2 = 1

전체 확률도 100%로 맞아요.
