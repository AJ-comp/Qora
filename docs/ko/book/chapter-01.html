1장. 큐비트 상태와 진폭에 대해 알아보자

한 큐비트 상태를 처음부터 어렵게 생각할 필요는 없어요.
먼저 “측정 결과값을 어떻게 상태 방향으로 적고, 그 방향 앞에 왜 수가 붙는지”를 순서대로 봐요.

1단계

측정 결과값에 상태 방향 이름을 붙여요

한 큐비트를 측정하면 결과값 0 또는 결과값 1이 나와요.
이 결과값을 그냥 숫자 0, 1로만 적을 수도 있을 것 같지만, 그렇게 쓰면 “측정 결과값”이라는 뜻만 보이고 상태 벡터의 기준 방향이라는 뜻은 잘 드러나지 않아요.
그래서 양자역학에서는 한 큐비트 상태를 벡터로 다룰 수 있도록, 측정 결과값 0에 대응하는 기준 상태를 |0⟩라고 쓰고 측정 결과값 1에 대응하는 기준 상태를 |1⟩라고 써요.

측정 결과값 0 → |0⟩
0 쪽 상태 방향
측정 결과값 1 → |1⟩
1 쪽 상태 방향

이 두 기호 |0⟩와 |1⟩를 한 큐비트의 계산 기저라고 불러요.

2단계

기준 방향 앞에는 성분의 크기를 나타내는 수가 붙어요

앞에서 |0⟩와 |1⟩를 상태 벡터의 기준 방향이라고 했어요.
이 말이 아직 낯설 수 있으니, 먼저 우리가 더 익숙한 평면 벡터로 같은 구조를 잠깐 확인해봐요.
평면 벡터는 x축 방향 성분과 y축 방향 성분으로 나누어 적을 수 있어요.

3·x방향 + 2·y방향

여기서 3은 x축 방향 성분의 계수이고, 2는 y축 방향 성분의 계수예요.
즉 벡터를 기준 방향으로 나누어 적으려면, 각 기준 방향 앞에 “그 방향 성분이 얼마나 있는지”를 나타내는 수가 붙어요.

3단계

큐비트 상태도 기준 방향별 성분으로 적어요

큐비트 상태에서는 |0⟩와 |1⟩가 기준 방향 역할을 해요.
그래서 각 방향 앞에 붙는 수가 필요하고, 그 수를 양자역학에서는 진폭이라고 불러요.

α|0⟩ + β|1⟩
α |0⟩ 방향 성분이 얼마나 들어 있는지 나타내는 진폭이에요.
β |1⟩ 방향 성분이 얼마나 들어 있는지 나타내는 진폭이에요.
|ψ⟩ = α|0⟩ + β|1⟩

따라서 이 식은 “이 큐비트 상태에는 0 쪽 진폭 α와 1 쪽 진폭 β가 함께 들어 있다”는 뜻이에요.

여기서 |ψ⟩는 이 큐비트의 상태 전체를 가리키는 기호예요.
ψ는 그리스 문자 프사이이고, |ψ⟩는 보통 켓 프사이 또는 프사이 상태라고 읽어요.
정식으로 말하면, |ψ⟩는 |0⟩ 방향의 진폭과 |1⟩ 방향의 진폭을 함께 담은 상태 벡터예요.
즉 큐비트 상태는 “나중에 0이 나올지 1이 나올지”라는 결과 하나가 아니라, 가능한 측정 결과값마다 붙어 있는 진폭 전체를 담고 있어요.

|0⟩와 |1⟩는 계산 기저 상태예요 |0⟩는 측정하면 결과값 0이 확실히 나오는 기준 상태이고, |1⟩는 측정하면 결과값 1이 확실히 나오는 기준 상태예요.
지금은 두 기호를 “0 쪽 방향”과 “1 쪽 방향”으로 읽어도 충분해요.
수학적으로는 |0⟩와 |1⟩가 한 큐비트 상태 공간의 서로 직교하는 두 기준 벡터 역할을 해요.
이번 장에서 잡은 것 한 큐비트 상태는 |0⟩ 방향 성분과 |1⟩ 방향 성분을 함께 담은 상태 벡터예요.
그리고 α와 β는 각 방향 성분 앞에 붙는 수이며, 양자역학에서는 이 수를 진폭이라고 불러요.
