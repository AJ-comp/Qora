3장. 중첩은 그냥 "모르는 상태"일까?

중첩은 양자컴퓨팅을 배울 때 가장 많이 오해되는 말이에요.
여기서는 중첩을 “결과를 아직 모른다”는 뜻이 아니라, 상태 벡터가 여러 계산 기저 방향의 진폭을 동시에 가진다는 뜻으로 읽을게요.

(|0⟩ + |1⟩) / √2

위 상태는 |0⟩ 방향의 진폭이 1/√2이고, |1⟩ 방향의 진폭도 1/√2인 상태예요.
두 계산 기저 방향이 모두 상태 벡터를 이루는 데 참여하므로 이 상태를 중첩 상태라고 불러요.

동전을 던진 뒤 손으로 덮어 두면, 동전은 이미 앞면이거나 뒷면으로 정해져 있고 우리는 그 결과를 모를 뿐이에요.
하지만 양자 중첩은 그런 고전적인 무지가 아니에요. 측정 전 상태 벡터 자체가 |0⟩ 성분과 |1⟩ 성분을 함께 가지고 있어요.

확률만 보면 상태를 다 알 수 없어요 양자 상태에는 측정 확률뿐 아니라 진폭의 부호와 위상도 들어 있어요.
이 정보는 바로 측정하면 사라져 보일 수 있지만, 다른 게이트를 지나면서 간섭을 만들기 때문에 실제 계산 결과에 영향을 줘요.

같은 50%처럼 보여도 정말 같은 상태일까?

상태 A

측정 결과값 1의 진폭이 양수인 상태

(|0⟩ + |1⟩) / √2

바로 측정하면 측정 결과값 0과 1이 각각 50% 확률로 나와요.

상태 B

측정 결과값 1의 진폭이 음수인 상태

(|0⟩ - |1⟩) / √2

바로 측정하면 이 상태 역시 측정 결과값 0과 1이 각각 50% 확률로 나와요.

두 상태는 바로 측정하면 같은 확률을 보이지만, 같은 상태는 아니에요.
측정 결과값 1의 진폭 부호가 다르고, 이 부호 차이가 다른 게이트를 한 번 더 적용할 때 결과 차이를 만들어요.
이 차이를 정석적으로 말하면 두 상태는 서로 다른 상태 벡터이고, H 게이트 같은 변환을 다시 만나면 서로 다른 방향으로 이동해요.
