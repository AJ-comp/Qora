Chapter 1. Learn About Qubit States and Amplitudes

A qubit state does not need to feel difficult from the first line.
Let's first follow, in order, how a measurement outcome gets written as a state direction and why a number is attached in front of that direction.

Step 1

Give each measurement outcome a state-direction name

When we measure one qubit, the result is either 0 or 1.
We could write those outcomes as the plain numbers 0 and 1, but then the notation only says “measurement outcome” and does not show that each outcome is also a basis direction of a state vector.
So in quantum mechanics, to treat a one-qubit state as a vector, we write the basis state corresponding to measurement outcome 0 as |0⟩, and the basis state corresponding to measurement outcome 1 as |1⟩.

measurement outcome 0 → |0⟩
state direction on the 0 side
measurement outcome 1 → |1⟩
state direction on the 1 side

These two symbols, |0⟩ and |1⟩, are called the computational basis for one qubit.

Step 2

A number in front of a basis direction tells us the size of that component

We just called |0⟩ and |1⟩ basis directions of a state vector.
If that still feels unfamiliar, first compare it with an ordinary vector on a plane.
A plane vector can be written by separating its x-direction component and y-direction component.

3·x-direction + 2·y-direction

Here, 3 is the coefficient of the x-direction component, and 2 is the coefficient of the y-direction component.
In other words, when we write a vector by basis directions, each direction needs a number that tells us how much component sits in that direction.

Step 3

A qubit state is written by components along its basis directions too

In a qubit state, |0⟩ and |1⟩ play the role of basis directions.
So we need a number in front of each direction, and in quantum mechanics that number is called an amplitude.

α|0⟩ + β|1⟩
α the amplitude that tells us how much |0⟩-direction component is in the state
β the amplitude that tells us how much |1⟩-direction component is in the state
|ψ⟩ = α|0⟩ + β|1⟩

So this formula means that the qubit state contains the 0-side amplitude α and the 1-side amplitude β together.

Here, |ψ⟩ is the symbol for the whole state of this qubit.
ψ is the Greek letter psi, and |ψ⟩ is usually read as ket psi or the psi state.
In formal language, |ψ⟩ is a state vector that holds both the amplitude in the |0⟩ direction and the amplitude in the |1⟩ direction.
So a qubit state is not one answer such as “will 0 or 1 come out later”; it stores the whole set of amplitudes attached to the possible measurement outcomes.

|0⟩ and |1⟩ are computational-basis states |0⟩ is the basis state that certainly gives measurement outcome 0, and |1⟩ is the basis state that certainly gives measurement outcome 1.
For now, it is enough to read them as the 0-side direction and the 1-side direction.
Mathematically, |0⟩ and |1⟩ act as two mutually orthogonal basis vectors in the one-qubit state space.
What we fixed in this chapter A one-qubit state is a state vector that contains both the |0⟩-direction component and the |1⟩-direction component.
The numbers α and β are attached in front of those components, and in quantum mechanics these numbers are called amplitudes.
