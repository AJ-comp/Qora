Chapter 4. Is Superposition Just a "State We Don't Know"?

Superposition is one of the most misunderstood words when learning quantum computing.
In quantum mechanics, superposition does not mean “we do not know the result yet.”
More precisely, once we choose a set of basis states, if a quantum state can be written as a linear combination with nonzero amplitudes on two or more of those basis states, we call it a superposition with respect to that basis.

That sentence can feel a little dense, so in this example we will use |0⟩ and |1⟩ as the basis states.
The state below has amplitude on the |0⟩ side and also amplitude on the |1⟩ side.

(|0⟩ + |1⟩) / √2

This expression is not a different story from the general form |ψ⟩ = α|0⟩ + β|1⟩ we saw in Chapter 1.
Dividing the whole parenthesized expression by √2 means multiplying each term inside the parentheses by 1/√2.

(|0⟩ + |1⟩) / √2
= (1/√2)(|0⟩ + |1⟩)
= (1/√2)|0⟩ + (1/√2)|1⟩

So, matched to the general form, this state has α = 1/√2 and β = 1/√2.

In the state above, the amplitude along the |0⟩ direction is 1/√2, and the amplitude along the |1⟩ direction is also 1/√2.

In this way, both basis states |0⟩ and |1⟩ take part in forming one state vector, so this state is a superposition with respect to the computational basis.

If you toss a coin and cover it with your hand, the coin is already fixed as heads or tails — we simply don't know the result.
But quantum superposition is not that kind of classical ignorance. Before measurement, the state vector itself holds both a |0⟩ component and a |1⟩ component.

Probabilities alone can't tell you the whole state A quantum state contains not only measurement probabilities but also the signs and phases of the amplitudes.
This information may seem to vanish if you measure right away, but it creates interference as the state passes through other gates, so it affects the actual result of the computation.

They both look like the same 50% — are they really the same state?

State A

A state where the amplitude of measurement outcome 1 is positive

(|0⟩ + |1⟩) / √2

If you measure right away, measurement outcomes 0 and 1 each appear with 50% probability.

State B

A state where the amplitude of measurement outcome 1 is negative

(|0⟩ - |1⟩) / √2

If you measure right away, this state also gives measurement outcomes 0 and 1 with 50% probability each.

Measured right away, the two states show the same probabilities, but they are not the same state.
The sign of the amplitude of measurement outcome 1 differs, and this sign difference produces different results the moment one more gate is applied.
Put more formally, the two states are different state vectors, and when they meet a transformation like the H gate again, they move in different directions.
