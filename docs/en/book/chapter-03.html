Chapter 3. Read the |0⟩ and |1⟩ States Directly

In Chapter 1, we read a single-qubit state like this.

|ψ⟩ = α|0⟩ + β|1⟩

In this chapter, we plug actual values into that formula.
We first calculate the |0⟩ state and the |1⟩ state, then read an even 50/50 state in the same way.

First, look at the |0⟩ state

The |0⟩ state has α = 1 and β = 0.
All of the amplitude sits on measurement outcome 0, and there is no amplitude on measurement outcome 1.

The substitution makes the notation much clearer.

|ψ⟩ = α|0⟩ + β|1⟩

For the |0⟩ state, α = 1 and β = 0, so the formula becomes this.

|ψ⟩ = 1|0⟩ + 0|1⟩

The term 0|1⟩ means there is no amplitude on the measurement outcome 1 side, so we can omit it.
The coefficient 1 in 1|0⟩ is usually omitted too, so we simply write |0⟩.

1|0⟩ + 0|1⟩ = |0⟩
Possible measurement outcomeAmplitude valueMeasurement probability
Measurement outcome 01|1|² = 100%
Measurement outcome 10|0|² = 0%

If you measure this state, measurement outcome 0 always appears.
In Qora, a newly prepared qubit starts in the |0⟩ state by default.

Try It in Qora

Measure a qubit right after preparing it

operation Main() {
    // Prepare one qubit
    // Right after preparation, q[0] is in the |0⟩ state
    // Amplitude of measurement outcome 0 = 1
    // Amplitude of measurement outcome 1 = 0
    use q = Qubit[1];

    // Measure q[0]
    // In an ideal execution environment: measurement outcome 0
    bit result = M(q[0]);
}

Now look at the |1⟩ state

The |1⟩ state has α = 0 and β = 1.
There is no amplitude on measurement outcome 0, and all of the amplitude sits on measurement outcome 1.

Putting α = 0 and β = 1 into the same formula gives this.

|ψ⟩ = 0|0⟩ + 1|1⟩

The term 0|0⟩ means there is no amplitude on the measurement outcome 0 side, so we can omit it.
The coefficient 1 in 1|1⟩ is usually omitted too, so we write |1⟩.

0|0⟩ + 1|1⟩ = |1⟩
Possible measurement outcomeAmplitude valueMeasurement probability
Measurement outcome 00|0|² = 0%
Measurement outcome 11|1|² = 100%

In Qora, applying an X gate to a qubit prepared in the |0⟩ state creates the |1⟩ state.
We will study the X gate in detail later, but for now you can read it as a gate that flips measurement outcome 0 and measurement outcome 1.

Try It in Qora

Create the |1⟩ state with the X gate

operation Main() {
    // Prepare one qubit
    // Right after preparation, q[0] is in the |0⟩ state
    use q = Qubit[1];

    // Apply the X gate
    // The |0⟩ state changes to the |1⟩ state
    X(q[0]);

    // In an ideal execution environment: measurement outcome 1
    bit result = M(q[0]);
}

Calculate an even 50/50 state the same way

Now look at a state where measurement outcomes 0 and 1 each appear with 50% probability.
We can write that state like this.

|ψ⟩ = (1/√2)|0⟩ + (1/√2)|1⟩

You do not need to know yet which gate creates this state.
For now, we only care about how to calculate measurement probabilities when the state is given.

Possible measurement outcomeAmplitude valueMeasurement probability
Measurement outcome 01/√2|1/√2|² = 1/2 = 50%
Measurement outcome 11/√2|1/√2|² = 1/2 = 50%
Remember the calculation order First read the values inside the state as amplitudes.
Then square the absolute value of each amplitude to get measurement probabilities.
We do not split probabilities first and then build the state. We read the amplitudes first, then calculate probabilities from them.
|0⟩ + |1⟩ and |01⟩: notation that is easy to confuse (|0⟩ + |1⟩) / √2 is a one-qubit state that carries amplitudes for measurement outcome 0 and measurement outcome 1 at the same time.
In contrast, |01⟩ is basis-state notation used when there are two qubits. It means the two-qubit state where the first qubit is 0 and the second qubit is 1.
