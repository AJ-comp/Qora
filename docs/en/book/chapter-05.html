Chapter 5. Why Must Quantum Gates Be Reversible?

A quantum gate cannot be just any linear transformation acting on the state vector.
In a standard quantum circuit, every gate before measurement must be a unitary transformation.
A unitary transformation preserves the length of the state vector, sends distinguishable states to distinguishable states, and has an inverse.

U†U = I

Here U is the gate matrix, and U† is its conjugate transpose.
The symbols may feel unfamiliar at first, but the core idea is simple: a unitary gate keeps the total probability of the state at 100%, and it never smears information away.

ConditionMeaningWhy it is needed
Length preservationThe total probability of the state vector stays at 1Measurement probabilities must always add up to 100%
Orthogonality preservationDistinguishable input states stay distinguishable in the outputDifferent inputs must not merge into the same output
Inverse existsThe gate can be undone, step for stepQuantum evolution before measurement loses no information

For example, a transformation that merges two different input states into the same output state is not a unitary transformation.
There is no way to recover the input from the output alone, because such a transformation squashes different directions of the state space into one.

Why Does the H Gate Need Its Minus Sign?

The H gate transforms the |0⟩ state and the |1⟩ state like this.

H gate(|0⟩) = (|0⟩ + |1⟩) / √2
H gate(|1⟩) = (|0⟩ - |1⟩) / √2

The important part here is the minus sign on the second line.
Thanks to this sign, the two output states (|0⟩ + |1⟩) / √2 and (|0⟩ - |1⟩) / √2 are orthogonal to each other.

H gate(|0⟩) = (|0⟩ + |1⟩) / √2
H gate(|1⟩) = (|0⟩ + |1⟩) / √2

Without the minus sign, the two input states |0⟩ and |1⟩ would merge into the same output state.
Then, looking at the output state alone, we could not tell what the input was, and we could not run the transformation backwards either.
So such a transformation cannot be accepted as a quantum gate before measurement.

Key point here The minus sign of the H gate is not a decoration added to make the story work out.
For the H gate to be a unitary gate, different input basis states must go to mutually orthogonal output states, and the minus sign is what creates that orthogonality.
H gate(H gate(|0⟩)) = |0⟩
