はじめに。なぜQoraで量子ゲートを学ぶのか？

量子コンピューティングを初めて学ぶと、キュービット、重ね合わせ、振幅、位相、干渉、もつれ、測定といった言葉が一度に出てきます。
これらの言葉は個別に見るともっともらしいのですが、回路の1行を実際に読もうとすると、すぐにあいまいになってしまいます。
そこでこの本では、言葉の意味を先に暗記するのではなく、小さな回路が状態ベクトルをどう変えるのかを、まず落ち着いて追いかけていきます。

たとえばHゲートを初めて見るとき、たいていは次のような式に最初に出会うはずです。

Hゲート(|0⟩) = (|0⟩ + |1⟩) / √2

この式は正しいのですが、初めて学ぶ人には疑問が一度にわいてきます。
|0⟩とは何か、なぜ|1⟩と足すのか、なぜ2ではなく√2で割るのか、そしてこの状態を測定すると何が出てくるのか、すべて確かめる必要があります。

この本で使う説明の順序

まずキュービットを状態ベクトルとして見て、その状態ベクトルの座標を振幅と呼ぶことにします。次に、振幅の絶対値の2乗が測定確率になるというBornの規則を確認し、ゲートを状態ベクトルに掛けるユニタリ行列として読んでいきます。

Qoraはここで2つの役割を担います。

第一に 量子回路を短く読みやすい形で書けるようにしてくれます。そのため、説明が抽象的な言葉だけにとどまりません。
第二に ゲートを適用する順序をコードとして固定してくれます。すると「どの状態でどんな変換が起きたのか」を1行ずつ確認できます。
数学で見るものQoraコードで見るもの読み方
初期状態 |0⟩use q = Qubit[1];キュービットを1つ、計算基底の|0⟩状態に準備します
ユニタリ変換 HH(q[0]);状態ベクトルにH行列を適用します
測定演算bit result = M(q[0]);現在の振幅から測定値を得ます
最初のQoraコード

1つのキュービットにHゲートを適用して測定してみよう

operation Main() {
    // キュービットを1つ準備
    use q = Qubit[1];

    // 最初のキュービットq[0]にHゲートを適用
    H(q[0]);

    // q[0]を測定して結果をresultに保存
    bit result = M(q[0]);
}

さあ、それでは各行がキュービットの状態と振幅をどう変えていくのか、順番に読んでいきましょう。
