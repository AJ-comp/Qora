Hゲート単独ドキュメント
Qoraで学ぶ量子ゲート / 概要

このドキュメントは、Qoraコードで量子回路を書いてみながら、そのコードが実際にどの量子力学の概念に触れているのかを順番に読んでいくものです。
目標はゲートの名前を暗記することではなく、1行のコードがキュービットの状態ベクトルの振幅をどう変えるのかを自分でたどれるようになることです。

このドキュメントの基準点

下の用語がまだ見慣れなくても大丈夫です。
今は、この本がどこへ向かうのかを示す標識として読んでください。
キュービットは複素数ベクトル空間の状態ベクトルとして扱い、測定確率はBorn規則で計算します。量子ゲートは状態ベクトルに作用するユニタリ線形変換として説明し、Qoraコードはその変換の順序を回路の形で書き留める道具として使います。

全体の流れ

ステップ1 状態をベクトルとして見る
1キュービットの状態は、|0⟩と|1⟩という計算基底に付いた2つの振幅で表現します。
ステップ2 振幅から確率を得る
振幅そのものが確率なのではなく、振幅の絶対値を2乗して初めて測定確率になります。
ステップ3 重ね合わせと干渉を先に理解する
振幅が複数の基底状態方向に置かれる状況と、振幅の寄与が再び出会って足し合わされたり打ち消し合ったりする状況を先に区別します。
ステップ4 ゲートを線形変換として読む
X、H、Z、CNOTのようなゲートが、状態ベクトルの振幅をどう変えるのかを計算します。
ステップ5 小さな回路を最後まで読む
複数のゲートが続くときに状態がどう変わり、最後の測定確率がどう決まるのかをたどります。
標準的な概念このドキュメントですることQoraで見える形
状態ベクトルキュービットがどんな振幅を持っているかを読みますuse q = Qubit[1];
ユニタリゲート状態ベクトルを保存できる形で変えますH(q[0]);, X(q[0]);
測定振幅の絶対値の2乗から結果の確率を得ますbit r = M(q[0]);
複数のキュービット基底状態の数が増え、ゲートがキュービットの間の関係を作りますCNOT(q[0], q[1]);
