3. 상태를 읽는 법 — 상태벡터와 진폭

기저 상태와 진폭 — 이후 모든 판정을 설명하는 틀

← 2. 컴파일러는 어떤 큐빗을 정리하는가 — 판별 사다리 코스 목차 4. 상태벡터를 큐빗별로 분해할 수 있는가 — 곱 상태와 얽힌 상태 →

앞 장의 판별 사다리에서 컴파일러는 청소할 큐빗을 세 질문으로 가려냈습니다.
앞의 두 질문은 큐빗이 누구의 것인지, 측정된 적이 있는지만 확인하면 답할 수 있었습니다.
그러나 마지막 질문, 곧 큐빗을 원래의 |0⟩로 되돌릴 수 있는지는 게이트가 큐빗의 상태를 어떻게 바꾸는지 읽을 수 있어야 답할 수 있습니다.

질문큐빗에게 묻는 것답하는 곳
0누구의 큐빗인가 (use로 만든 앤실라인가)2장
1측정된 적이 있는가2장
2되돌릴 수 있는가9장부터

그래서 이 장에서는 그 마지막 질문에 답하기 위한 바탕, 곧 상태를 읽는 틀부터 세웁니다.
이후 장들의 모든 판정, 예컨대 "이 게이트를 거친 앤실라는 왜 되돌릴 수 없는가"라는 질문은 이 틀 하나로 전부 설명됩니다.

그 틀의 출발점은 양자 상태의 형식적 정의입니다.
큐빗 여러 개로 이루어진 기계의 양자 상태는, 모든 큐빗의 값 조합 하나하나를 나타내는 기저 상태들의 선형 결합이며, 결합에 참여하는 각 기저 상태에는 복소수 진폭이 하나씩 붙습니다.

예를 들어 큐빗이 둘이면 값 조합은 |00⟩·|01⟩·|10⟩·|11⟩ 네 가지이고, 양자 상태는 이 네 기저 상태를 각자의 진폭으로 섞은 선형 결합입니다.
이를 식으로 쓰면 다음과 같습니다.

양자 상태  =  c₀|00⟩  +  c₁|01⟩  +  c₂|10⟩  +  c₃|11⟩
네 기저 상태를 더한 선형 결합이고, 각 기저 상태 앞의 c₀…c₃가 그 기저 상태의 복소수 진폭입니다.
큐빗이 n개이면 기저 상태(항)는 2ⁿ개입니다.
지금은 큐빗이 2개라 4개여서, 진폭이 c₀부터 c₃까지인 것입니다.
각 진폭의 절댓값 제곱 |cᵢ|²은 그 기저 상태가 측정될 확률이므로, 넷을 모두 더하면 항상 1입니다 (|c₀|² + |c₁|² + |c₂|² + |c₃|² = 1).

이 정의를 표 한 장으로 옮겨 적은 것이 아래의 상태벡터입니다.
위 식에서 c₀|00⟩, c₁|01⟩처럼 더해지던 각 부분이 표에서는 한 줄이 됩니다.

항 (기저 상태)진폭
|00⟩c₀
|01⟩c₁
|10⟩c₂
|11⟩c₃

이 표의 한 줄을 항이라 부르고, 한 항이 기저 상태 하나에 대응합니다.
1장에서 상태가 나뉘며 생긴 기저 상태 하나가 여기서는 항 하나입니다.

예를 들어 아래 프로그램이 상태벡터를 어떻게 만들어 가는지 한 단계씩 따라가 보겠습니다.
그 전에, 이 장에서 쓰는 두 자리 켓 |b,a⟩를 읽는 법부터 한 번 보고 갑시다.

|0,0⟩
앞 자리 = b의 값  ·  뒤 자리 = a의 값
예를 들어 |0,1⟩ 은 b=0, a=1.  (쉼표 없이 |01⟩로도 씁니다.) 
use a = Qubit[1];  use b = Qubit[1];
H(a[0]);             // a가 0과 1의 두 항으로 나뉨
CNOT(a[0], b[0]);    // b가 a의 값을 기록
각 켓은 |b,a⟩ — 앞자리가 b, 뒷자리가 a
|0,0⟩
시작 — 두 큐빗 모두 |0⟩
H(a) a를 0·1 두 갈래로
↓
(|0,0⟩ + |0,1⟩)/√2
새로 생긴 |0,1⟩은 앞자리 b=0, 뒷자리 a=1
CNOT b가 a의 값을 복사
↓
(|0,0⟩ + |1,1⟩)/√2
a=1이던 |0,1⟩에서 앞자리 b가 0→1 → |1,1⟩ (두 항 모두 b=a)

이렇게 b가 a를 따라가므로, b≠a인 |0,1⟩·|1,0⟩ 항은 아예 생기지 않고 |0,0⟩·|1,1⟩ 두 항만 남습니다.

이 최종 상태를 상태벡터 표로 쓰면 아래와 같습니다.

항b 값a 값진폭
|0,0⟩001/√2
|1,1⟩111/√2

이 표에서 한 항을 읽을 때 헷갈리기 쉬운 점은 두 가지입니다.
① 한 항의 값은 기계 전체의 값 조합 하나입니다. 기저 상태는 특정 큐빗의 값이 아니라 모든 큐빗의 값을 한 세트로 묶은 것이므로, 한 항에 b값과 a값이 함께 적힙니다.
② 진폭은 항마다 하나뿐입니다. "b의 진폭"과 "a의 진폭"이 따로 있는 것이 아니라, 진폭 하나가 그 값 조합 전체에 붙습니다.
이 둘을 식으로 쓰면 표의 한 행은 c|b,a⟩ 꼴입니다.

표의 한 행  =  c|b,a⟩
|b,a⟩는 기저 상태(규칙 ①), c는 진폭(규칙 ②)입니다.
예를 들어 첫 항은 1/√2|0,0⟩입니다.
