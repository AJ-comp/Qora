7. 전역 위상과 상대 위상 — 측정으로 확인하기

같은 Z 게이트가 항 1개엔 전역 위상, 항 2개엔 상대 위상 — 측정으로 확인합니다

앞 장에서 위상이 항의 속성이라는 것과, 위상이 모든 항에 똑같이 붙으면 전역 위상(측정에 안 드러남), 항마다 다르게 붙으면 상대 위상(측정에 드러남)이 된다는 것을 보았습니다.
그 “측정에 드러난다”가 말뿐이 아니라 실제로 그런지는 아직 직접 보지 않았습니다.
이 장에서는 Z 게이트 하나로, 같은 게이트가 항이 하나뿐인 상태에서는 전역 위상이 되어 결과를 그대로 두고, 항이 두 개인 상태에서는 상대 위상이 되어 결과를 뒤집는 과정을 단계별로 확인합니다.

확인에 쓸 게이트는 Z입니다. Z 게이트는 값이 1인 항의 진폭에 −1을 곱하고 기저 값은 건드리지 않습니다.
같은 Z 게이트를 항 개수만 다른 두 상태벡터에 한 번씩 적용해, 언제 전역 위상이 되고 언제 상대 위상이 되는지 봅니다.

1단계 — 항이 하나뿐인 상태벡터: Z의 부호는 전역 위상이 됩니다

use q = Qubit[1];
X(q[0]);       // 상태벡터: (1)→1, 줄 하나
Z(q[0]);       // 유일한 줄에 부호 − → 전역 위상
bit m = M(q[0]);
// m = 1 (확률 100%) — Z를 지워도 동일
시점상태벡터 (q)일어난 일
X 후(1) ×1항 하나짜리 상태벡터
Z 후(1) ×−1유일한 항 = 모든 항에 부호 − → 전역 위상
M 후m = 1 (확률 100%)확률은 진폭 크기의 제곱 |−1|² = 1 — Z 게이트가 없을 때와 동일

부호가 상태벡터의 모든 항에 붙었고, 측정 확률은 진폭의 크기만 보기 때문에 결과는 Z 게이트가 없을 때와 완전히 같습니다.
전역 위상은 이렇게 어떤 측정으로도 흔적을 남기지 않습니다.

2단계 — 항이 두 개인 상태벡터: 같은 Z가 상대 위상이 되어 결과를 뒤집습니다

이번에는 1장의 기준선이었던 "H 두 번, 결과는 항상 0" 프로그램을 다시 가져와, 두 H 사이에 같은 Z 한 줄만 끼워 넣습니다. 

use q = Qubit[1];
H(q[0]);       // 상태벡터: (0)→1/√2, (1)→1/√2
Z(q[0]);       // ← 추가된 한 줄: 1인 줄에만 부호 −
H(q[0]);
bit m = M(q[0]);
// m = 1 (확률 100%) — Z가 없으면 m = 0
시점상태벡터 (q)일어난 일
첫 H 후(0) ×1/√2 · (1) ×1/√2항이 둘로 나뉨
Z 후(0) ×1/√2 · (1) ×−1/√21인 항에만 부호 − → 상대 위상
둘째 H 후(1) ×10으로 모이는 기여 +1/2와 −1/2가 상쇄, 1로 모이는 기여 +1/2와 +1/2가 합쳐짐
M 후m = 1 (확률 100%)Z가 없으면 두 항이 (0) ×1로 합쳐져 m = 0

같은 Z 게이트가 이번에는 두 항 중 1인 항에만 부호를 붙였습니다.
둘째 H 게이트는 각 항을 다시 두 항으로 나눠 같은 값끼리 합치는데, 0으로 모이는 두 기여는 +1/2와 −1/2라서 상쇄되고 1로 모이는 두 기여는 +1/2와 +1/2라서 살아남습니다.
그래서 측정 결과가 "항상 0"에서 "항상 1"로 확실하게 뒤집힙니다.
1단계와 프로그램 두 개를 나란히 두고 보면, 전역 위상과 상대 위상의 차이가 측정으로 직접 관측된다는 사실이 확인됩니다.
