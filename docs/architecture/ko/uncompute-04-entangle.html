4. 상태벡터를 큐빗별로 분해할 수 있는가 — 곱 상태와 얽힌 상태

두 큐빗의 상태를 따로 적을 수 있는지 확인하여 곱 상태와 얽힌 상태를 구분합니다

← 3. 상태를 읽는 법 — 상태벡터와 진폭 코스 목차 5. 곱에서 얽힘으로 — 프로그램 단계별 확인 →

3장에서는 상태벡터를 계산 기저의 선형 결합으로 읽고, 전체 상태벡터의 각 진폭이 해당 기저 상태 앞에 곱해지는 계수임을 확인했습니다.
이제 두 큐빗의 전체 상태벡터를 큐빗 b의 상태벡터와 큐빗 a의 상태벡터로 분해할 수 있는지 살펴보겠습니다.

이 분해가 가능하면 두 큐빗을 각각의 순수 상태벡터로 기술할 수 있지만, 가능하지 않으면 독립된 순수 상태벡터 두 개로 나누어 기술할 수 없습니다.
바로 이 차이가 곱 상태와 얽힌 상태를 구분합니다.
먼저 쉬운 기준과 대각선 곱 비교로 차이를 확인한 뒤, 그 판정법이 성립하는 이유를 공식으로 정리하겠습니다.

먼저 쉽게

두 큐빗을 따로 적어도 원래 상태를 다시 만들 수 있는가

큐빗 b의 상태와 큐빗 a의 상태를 각각 따로 적은 뒤, 두 상태를 합쳐 원래의 전체 상태를 정확히 다시 만들 수 있는지 생각해보겠습니다.
다시 만들 수 있으면 두 큐빗을 따로 기술할 수 있지만, 아무리 나누어 적어도 원래 상태를 재현할 수 없다면 두 큐빗을 하나의 전체 상태로 다루어야 합니다.

쉬운 기준 큐빗별 상태를 따로 적고 다시 합쳐 원래 상태를 만들 수 있으면 곱 상태이고, 만들 수 없으면 얽힌 상태입니다.

빠른 판정

네 진폭을 놓고 두 대각선의 곱을 비교합니다

두 큐빗의 상태가 곱 상태인지 얽힌 상태인지 확인할 때마다 큐빗별 상태를 직접 찾아낼 필요는 없습니다.
이 장에서 다루는 두 큐빗 순수 상태에서는 네 기저 상태의 진폭을 2×2 자리에 놓고 두 대각선의 곱만 비교하면 됩니다.

기저 상태 |ba⟩의 첫째 자리는 큐빗 b, 둘째 자리는 큐빗 a를 나타냅니다.
각 기저 상태의 진폭을 차례로 α, β, γ, δ라고 하겠습니다.

|ψ⟩= α|00⟩+ β|01⟩+ γ|10⟩+ δ|11⟩
αδ=βγ
곱 상태 — 큐빗별 상태벡터로 분해할 수 있습니다.
αδ≠βγ
얽힌 상태 — 큐빗별 상태벡터로 분해할 수 없습니다.

앞의 두 상태에 바로 적용하기

이제 그림에서 본 곱 상태와 얽힌 상태에 진폭을 대입해보겠습니다.

곱 상태 예제와 얽힌 상태 예제의 두 대각선 곱 비교
상태↘ αδ↙ βγ비교판정
(|00⟩+|10⟩)/√2 (1/√2) × 0 = 0 0 × (1/√2) = 0 0 = 0 곱 상태
(|00⟩+|11⟩)/√2 (1/√2) × (1/√2) = 1/2 0 × 0 = 0 1/2 ≠ 0 얽힌 상태

첫 번째 상태는 두 대각선 곱이 모두 0이므로 곱 상태이고, 실제로 다음과 같이 큐빗별 상태로 나눌 수 있습니다.

|00⟩+|10⟩2 = |0⟩+|1⟩2b ⊗ |0⟩a

반면에 두 번째 상태는 두 대각선 곱이 1/2과 0으로 다르므로 얽힌 상태입니다.
여기까지는 판정법을 눈으로 확인한 단계입니다.
이제 이 비교가 텐서 곱 분해 여부와 정확히 일치하는 이유를 공식으로 확인하겠습니다.

정확한 정의

판정 결과를 텐서 곱으로 정의합니다

앞에서 두 큐빗의 상태를 따로 적고 다시 합친다고 설명했습니다.
이때 두 부분계의 상태벡터를 하나의 전체 상태벡터로 결합하는 연산을 텐서 곱이라고 하며, 기호 ⊗로 나타냅니다.

|ψ⟩ba = |φ⟩b ⊗ |χ⟩a

여기서 |φ⟩b와 |χ⟩a는 각각 큐빗 b와 큐빗 a의 단일 큐빗 상태벡터입니다.
텐서 곱은 두 수를 단순히 곱하는 연산이 아니라, 두 상태벡터에 들어 있는 모든 기저 상태 조합과 그 진폭을 함께 전개합니다.

곱 상태
위 등식을 만족하는 두 단일 큐빗 상태벡터가 존재하는 상태입니다.
따라서 전체 상태벡터를 큐빗별 상태벡터의 텐서 곱으로 분해할 수 있습니다.
얽힌 상태
그러한 두 상태벡터가 존재하지 않는 상태입니다.
따라서 전체 상태벡터를 큐빗별 상태벡터의 텐서 곱으로 분해할 수 없습니다.

측정 결과가 항상 같거나 다르다는 현상은 얽힘의 정의가 아닙니다.
예를 들어 (|01⟩+|10⟩)/√2는 계산 기저로 측정할 때 두 결과가 항상 다르지만, 텐서 곱으로 분해되지 않으므로 얽힌 상태입니다.
핵심은 측정 결과의 모양이 아니라 전체 상태벡터의 텐서 곱 분해 가능성입니다.

공식 전개

왜 두 대각선의 곱을 비교하는가

앞에서는 두 대각선의 곱이 같은지만 확인했습니다.
이제 곱 상태의 텐서 곱을 직접 전개하여 그 판정법이 성립하는 이유를 확인하겠습니다.
큐빗 b와 큐빗 a의 단일 큐빗 상태벡터를 각각 다음과 같이 나타내겠습니다.
여기서 u₀과 u₁은 큐빗 b의 진폭이고, v₀과 v₁은 큐빗 a의 진폭입니다.

|φ⟩b = u0|0⟩+u1|1⟩ |χ⟩a = v0|0⟩+v1|1⟩

두 상태벡터의 텐서 곱을 전개하면 네 기저 상태의 진폭이 다음과 같이 정해집니다.

(u0|0⟩+u1|1⟩) ⊗ (v0|0⟩+v1|1⟩) = u0v0|00⟩ +u0v1|01⟩ +u1v0|10⟩ +u1v1|11⟩

이 전개식을 앞에서 사용한 α, β, γ, δ와 비교하면 각 진폭의 관계가 다음과 같이 정해집니다.

α=u0v0, β=u0v1, γ=u1v0, δ=u1v1 αδ = (u0v0) (u1v1) = u0u1v0v1 = (u0v1) (u1v0) = βγ

전개의 결론 곱 상태라면 αδ와 βγ가 반드시 같음을 확인했습니다.
반대 방향도 성립하는지는 다음 절에서 계수 행렬을 통해 확인합니다.

전문적 해석

두 대각선 곱의 차이는 계수 행렬의 행렬식입니다

빠른 판정에서 네 진폭을 놓았던 2×2 배열을 선형대수에서는 계수 행렬이라고 합니다.
행은 큐빗 b의 값, 열은 큐빗 a의 값에 대응하며, 이 계수 행렬을 C라고 하겠습니다.

C= αβ γδ

2×2 계수 행렬에서 두 대각선 곱의 차이 αδ−βγ를 행렬식이라고 합니다.
따라서 앞에서 두 대각선 곱이 같은지 비교한 것은 행렬식이 0인지 확인한 것과 같습니다.

det⁡C = αδ−βγ

앞의 텐서 곱 전개는 곱 상태라면 행렬식이 0이라는 방향을 보여주었습니다.
반대 방향도 성립합니다.
행렬의 랭크는 서로 독립인 행 또는 열의 수를 나타냅니다.
정규화된 상태의 계수 행렬은 영행렬이 아니므로, 행렬식이 0이면 랭크가 1입니다.
랭크가 1인 2×2 행렬은 한 열벡터와 한 행벡터의 곱으로 분해되며, 이는 곧 두 단일 큐빗 상태벡터의 텐서 곱에 해당합니다.
그러므로 다음 조건은 두 큐빗 순수 상태에서 곱 상태를 판별하는 필요충분조건입니다.

αδ=βγ ⇔ det⁡C=0 ⇔ rank⁡C=1 ⇔ 곱 상태

반대로 αδ≠βγ이면 행렬식이 0이 아니고 계수 행렬의 랭크가 2이므로, 두 단일 큐빗 상태벡터의 텐서 곱으로 분해할 수 없는 얽힌 상태입니다.

적용 범위
이 대각선 곱 비교는 두 큐빗 순수 상태에 직접 적용하는 판정법입니다.
세 큐빗 이상에서는 어떤 부분계 분할을 기준으로 분해 가능성을 검사하는지 명시해야 하며, 완전한 곱 상태와 일부 부분계만 분리되는 상태도 구분해야 합니다.
다만 주어진 분할에 대해 텐서 곱으로 분해되지 않으면 얽힌 상태라는 원리는 같습니다.

장 요약

상태벡터의 분해 가능성이 얽힘을 결정합니다

두 큐빗의 전체 상태벡터를 두 단일 큐빗 상태벡터의 텐서 곱으로 분해할 수 있으면 곱 상태이고, 분해할 수 없으면 얽힌 상태입니다.
두 큐빗 순수 상태에서는 진폭 네 개를 2×2 자리에 놓고 두 대각선의 곱을 비교하여 이를 빠르게 판별할 수 있습니다.
두 곱이 같다는 조건 αδ=βγ를 전문적으로 표현하면 계수 행렬의 행렬식이 0이고 랭크가 1이라는 뜻입니다.
